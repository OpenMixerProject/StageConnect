# StageConnect
This repository contains documentation and the API for Behringers StageConnect.

StageConnect is based on the automotive A²B and is used to connect the Behringer WING to the Midas DP48, for instance. Analog Devices [A²B audio](https://www.analog.com/en/solutions/a2b-audio-bus.html) bus is used as the technology basis and uses simple XLR connectors between two devices.
